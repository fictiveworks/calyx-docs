---
title: Modifiers
---

## uppercase

```
{expansion.uppercase}
```

Converts a template expansion to uppercase.

## lowercase

```
{expansion.lowercase}
```

Converts a template expansion to lowercase.

## titlecase

```
{expansion.titlecase}
```

Converts a template expansion to title case.

## sentencecase

```
{expansion.sentencecase}
```

Converts a template expansion to sentence case.
